Yii Widget class for credit-card form, which can be added into another view

To use this widget in an action:

(1) the Model being used by the action must extend DynamicModelBase (or any of it's descendants), and add CreditCardFormSegmentModel's attributes on construction. Example:

class AnotherFormModel extends DynamicModelBase { public function __construct() { parent::__construct(); // add members from CreditCardFormSegmentModel $this->addMixinModel(new CreditCardFormSegmentModel()); }

(2) The widget is invoked in the action's view. Example:

$form = $this->beginWidget('CActiveForm', array('id'=>'AnotherForm')); $this->widget('application.widgets.credit_card_payment.CreditCardFormSegmentWidget', array('form'=> $form, 'model'=> $model)); $this->endWidget();
