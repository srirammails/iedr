jqGridComandButtonsWidget

Description
Description | Vars (details) | Methods (details)

Widget which adds command buttons to a jqGrid, so that clicking on one of the buttons then causes the posting of all the selected row keys to the specified action-url along with additional 'command' arguments.

Example Usage:

$this->widget('application.widgets.jqGrid.jqGridComandButtonsWidget', array( 'pk_column_index' => 1, 'actionurl' => $this->createUrl('mycontroller/transmogrify'), 'returnurl' => $this->createUrl('mycontroller/list_widgets'), 'invoiceType' => $model->invoiceType, 'commandlist' => array( 'transmog' => 'Transmogrify %n Widgets', ), ));
